GitHub PR Merge Email Check Settings

Enter a regular expression pattern to validate commit email addresses. The "Confirm merge" button will be disabled if the selected email doesn't match this pattern.
Examples:
• .*@example\.com$ - Matches any email ending with @example.com
• ^(user1|user2)@.* - Matches emails starting with user1@ or user2@
• .*@(company1|company2)\.com$ - Matches emails from company1.com or company2.com
